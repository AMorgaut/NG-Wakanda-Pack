NG-Wakanda-Pack

This solution pack contain several demo pages to learn how to use angular-wakanda

All this pack and its running examples are available from this Github repository: NG-Wakanda-Pack

1 - Integrate angular-wakanda in your project

Thanks to Bower, Wakanda is very easy to integrate into Angular interfaces

Even manually, angular-wakanda only requires a single JS file to include.

2 - Initialize the Wakanda Service

The Wakanda Service init(dataClasses) method lets you choose to load all the available Model (default) or only specified DataClasses

It works as a JavaScript Promise. Let's load everything.

angular-wakanda lets you have all your Wakanda Model usable directly from Angular based Web Application interfaces.
